Frequently Asked Questions

Everything you need to know about AutoContext AI

How does the text input work?

+

Simply type your request in the text box and click send. Our AI will analyze your input along with the webpage context to provide intelligent and relevant responses.

Is my data secure?

+

Yes! We use Google's secure OAuth system for authentication. Your API keys are stored locally in your browser using Chrome's secure storage. We don't collect or store your personal data.

What happens when I reach the daily limit?

+

Free users get 10 requests per day. Once you reach this limit, you can upgrade to Basic or Pro for more requests, or wait until the next day when your limit resets.

Do I need an OpenAI API key?

+

No! We handle all the API keys for you. Simply sign in and start using the AI immediately. No setup or configuration required.

Can I cancel my subscription?

+

Absolutely! You can cancel your subscription at any time. You'll continue to have access until the end of your current billing period, then return to the free plan.

Does it work on all websites?

+

Yes! AutoContext AI works on any website. It can read the page content and provide context-aware responses based on what you're viewing.
